# DATANORM Parser
Datanorm -> CSV (; separated and stored as txt)

Allowed set order:
 + T -> A -> P

## Execution
`gcc datanormparser.c -o dnp`

`./dnp [filename1] [filename2] ...`

### Example
`gcc datanormparser.c -o dnp`

`./dnp datanorm.001.html datanorm.006.html datpreis.006.html`
